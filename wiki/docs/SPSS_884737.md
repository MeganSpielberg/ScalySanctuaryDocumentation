# SPSS

## 💡 Why SPSS

IBM SPSS Statistics was used for all quantitative analysis in this
project. It was recommended by project coach Wouter Boendermaker, and
chosen over alternatives like R or Python/pandas for a few reasons:

- **Recommended by the coach:** Direct guidance to use SPSS for the
  study's statistical analysis.

- **Learning material availability:** Extensive course material and
  the reference book *Discovering Statistics Using IBM SPSS Statistics*
  (listed in [Materials](Materials_983041.md)) were built around SPSS,
  making it the most supported option for learning the required tests.

- **Established standard in research:** SPSS is a long-standing,
  widely recognized tool in behavioral and social science research,
  which made it a safe choice for a study intended to produce
  academically defensible results.

## 📊 How It Was Used

SPSS was used to run all inferential statistics for the study,
including outlier analysis, a mixed factorial repeated measures ANOVA,
a one-way ANOVA on engagement scores, and Pearson correlations between
knowledge gain, engagement, and LLM interaction frequency.

The full breakdown of each analysis - rationale, step-by-step SPSS
procedure, and results - is documented in
[Data Analysis](Data-Analysis_950273.md).
