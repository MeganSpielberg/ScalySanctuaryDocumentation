# Godot Plugins

Plugins made by abstracting functionality implemented and used in the
Scaly Sanctuary research prototype, and published for reuse by other
Godot projects.

## 🤖 Godot LLM Plugin

Wraps the LLM gateway/fallback integration built for the AI veterinary
NPC (see [Devlog: LLM Integration](6291458.md) and
[Devlog: LLM Host Failsafe](3899400.md)) into a reusable plugin.

- GitHub: [https://github.com/Tinoyipman/GodotLLM_Plugin](https://github.com/Tinoyipman/GodotLLM_Plugin)
- Godot Asset Library listing: [https://godotengine.org/asset-library/asset/5200](https://godotengine.org/asset-library/asset/5200)

## 🖱️ Godot Play Interaction Logger Plugin

Wraps the player-interaction logging system built for the research data
collection pipeline (see [Devlog: Research Data Logging System](3735560.md))
into a reusable plugin.

- GitHub: [https://github.com/Tinoyipman/GodotPlayInteractionLogger_Plugin](https://github.com/Tinoyipman/GodotPlayInteractionLogger_Plugin)
- Godot Asset Library listing: [https://godotengine.org/asset-library/asset/5208](https://godotengine.org/asset-library/asset/5208)
